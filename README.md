# 🐵 Monkey Hunter

Um jogo 2D desenvolvido com **HTML, CSS e JavaScript**, focado em prática de **Programação Orientada a Objetos (POO)** e lógica de jogos.

---

## 🎮 Sobre o jogo

No **Monkey Hunter**, você controla um personagem que deve sobreviver e derrotar inimigos enquanto acumula pontos.

O jogo conta com:

- Sistema de fases
- Inimigos com aumento de dificuldade
- Sistema de vida
- Itens coletáveis
- Boss com barra de vida
- Modo 1 jogador e 2 jogadores

---

## 🕹️ Controles

### 👤 Player 1 (Marrom)
- `W` `A` `S` `D` → Movimentação
- `F` → Atirar

### 👤 Player 2 (Azul)
- `↑ ↓ ← →` → Movimentação
- `ENTER` → Atirar

---

## 🎯 Objetivo

- Derrotar inimigos 🐒  
- Sobreviver o máximo possível  
- Derrotar o **Boss final** 💀  
- Fazer a maior pontuação 🎯  

---

## ⚙️ Modos de jogo

O jogo possui dois modos:

### 🔹 1 Player
- Apenas o Player 1 é ativo
- Player 2 é totalmente desativado

### 🔹 2 Players
- Ambos os jogadores podem jogar simultaneamente
